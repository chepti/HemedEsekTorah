# העלאה לוורדפרס

בתיקייה הזו נמצאת חבילת התוסף.

**מה לארוז ולהעלות:** רק את התיקייה **`hemed-esek-torah`** (זו שבתוך התיקייה הזו) — לא את `wordpress-plugin` ולא את שורש המאגר.

אותה תיקייה מעלים דרך «תוסף חדש» או מעתיקים ל־`wp-content/plugins/hemed-esek-torah/`.

אל תכללי את שאר מאגר הפיתוח (שורש הפרויקט מעל `wordpress-plugin`).
