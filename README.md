# Hemed Esek Torah

תוסף WordPress לשיתוף פעילויות "עסק תורה" בחמ"ד.

## דרישות

- WordPress 6.0 ומעלה
- PHP 7.4 ומעלה
- ACF חינמי פעיל באתר

## התקנה

במאגר, כל קבצי התוסף נמצאים בתוך התיקייה `hemed-esek-torah/` — **אותה תיקייה** מעתיקים ל־`wp-content/plugins/`.

1. להעתיק את **כל** התיקייה `hemed-esek-torah` (עם השם הזה) אל `wp-content/plugins/`, כך שהנתיב לקובץ הראשי יהיה בדיוק:  
   `wp-content/plugins/hemed-esek-torah/hemed-esek-torah.php`
2. לא לשים את התיקייה בתוך תת־תיקייה נוספת תחת `plugins` (אחרת וורדפרס לא ימצא את קובץ ה־PHP בשורש התוסף).
3. להפעיל את התוסף דרך מסך התוספים.
4. להפעיל את ACF אם הוא עדיין לא פעיל.
5. לגשת אל הגדרות > מבנה קישורים וללחוץ שמירה.

אם מופיעה השגיאה **קובץ התוסף לא קיים**: לבדוק בשרת שהנתיב הוא `.../plugins/hemed-esek-torah/hemed-esek-torah.php` ולא `.../plugins/משהו-אחר/hemed-esek-torah/hemed-esek-torah.php` בגלל אריזה כפולה.

## כתובות עמוד פעילות (מגרסה 1.0.3)

כתובת עמוד פעילות מוצגת כ־**מזהה מספרי** (יציב בכל השרתים, בלי slug בעברית בנתיב):

- עמוד בודד: `https://דומיין/esek-torah/p/123/` (123 הוא מזהה הפוסט)
- ארכיון כל הפעילויות: `https://דומיין/esek-torah/`

אם נשארו קישורים ישנים מהצורה `.../esek-torah/שם-בעברית/`, התוסף מנסה **הפניה 301** לכתובת החדשה כשהשם תואם לפוסט.

לאחר עדכון התוסף: **הגדרות → מבנה קישורים → שמירה** (פעם אחת), או שהריענון יתבצע אוטומטית כשמספר הגרסה משתנה.

## 404 על עמוד פעילות (מידע כללי)

1. **מבנה קישורים:** גשי ל־**הגדרות → מבנה קישורים** ולחצי **שמירת שינויים** (גם בלי לשנות כלום).
2. **עמוד קבוע:** אם קיים **עמוד** באתר עם כתובת `esek-torah`, הוא עלול להתנגש — שני שינוי שם העמוד או שינוי הבסיס בתוסף (דורש מפתח).

## העיצוב לא נטען (טפסים וכפתורים ללא CSS)

ה־CSS וה־JS נטענים ב־`wp_enqueue_scripts` כדי שיופיעו ב־`<head>` בזמן. אם עדיין אין עיצוב: לוודא שבשרת קיימים `assets/css/frontend.css` ו־`assets/js/frontend.js` בתוך תיקיית התוסף. אם השורטקוד רק בתבנית (לא בתוכן העמוד), אפשר להוסיף בתבנית: `add_filter( 'het_force_enqueue_assets', '__return_true' );`

## שימוש

בדף הבית מומלץ להכניס את השורטקוד:

```text
[hemed_esek_torah_home]
```

השורטקוד מציג גריד פעילויות, חיפוש וסינון, וכפתור שפותח טופס העלאה בחלונית.

אפשר גם להשתמש בנפרד:

```text
[hemed_esek_torah_grid]
[hemed_esek_torah_submit]
```

## שדות ACF

התוסף רושם אוטומטית קבוצת שדות עבור סוג התוכן "פעילויות עסק תורה":

- סמל מוסד - 6 ספרות.
- שם בית הספר.
- מחוז.
- שלב גיל.
- בנים / בנות / מעורב.
- סוג פעילות.
- פירוט הפעילות.
- צ׳קליסט הכנות.
- 5 שדות תמונה.
- 5 קבצי עזר ו־5 קישורי עזר.
- שותפים.
- ציטוטי משובים.

## אישור תכנים

שליחה מהטופס הקדמי יוצרת פעילות בסטטוס `ממתין לאישור`. עורכים ומנהלים רואים פעמון בסרגל הניהול עם מספר הפעילויות שממתינות לאישור.

## לוגו

התוסף משתמש בלוגו האתר שמוגדר בהתאמה אישית של WordPress. אם אין לוגו מוגדר, התוסף מציג fallback לפי לוגו חמ"ד שסופק.
