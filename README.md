# Hemed Esek Torah

תוסף WordPress לשיתוף פעילויות "עסק תורה" בחמ"ד.

## דרישות

- WordPress 6.0 ומעלה
- PHP 7.4 ומעלה
- ACF חינמי פעיל באתר

## התקנה

וורדפרס מחפש את קובץ ה־PHP הראשי **ישירות** בתוך `wp-content/plugins/שם-התיקיה/` (לא בתוך תת־תיקייה נוספת). אם מעלים ZIP מהריפו, חשוב שבתוך ה־ZIP יופיעו `hemed-esek-torah.php`, `includes/` ו־`assets/` בשורש הארכיון.

1. **דרך מסך התוספים:** ליצור ZIP שבשורשו הקבצים שלמעלה, או לפרוס את תוכן שורש הריפו לתיקייה אחת תחת `wp-content/plugins/hemed-esek-torah/` (שם התיקייה לבחירתכם).
2. להפעיל את התוסף דרך מסך התוספים.
3. להפעיל את ACF אם הוא עדיין לא פעיל.
4. לגשת אל הגדרות > מבנה קישורים וללחוץ שמירה.

אם מופיעה השגיאה **קובץ התוסף לא קיים** (או באנגלית *The plugin file does not exist*): כנראה שהתיקייה בשרת נראית כמו `.../plugins/משהו/hemed-esek-torah/hemed-esek-torah.php` — צריך להעלות כך שהנתיב יהיה `.../plugins/hemed-esek-torah/hemed-esek-torah.php` (רמה אחת פחות של תיקיות).

## שימוש

בדף הבית מומלץ להכניס את השורטקוד:

```text
[hemed_esek_torah_home]
```

השורטקוד מציג גריד פעילויות, חיפוש וסינון, וכפתור שפותח טופס העלאה בחלונית.

אפשר גם להשתמש בנפרד:

```text
[hemed_esek_torah_grid]
[hemed_esek_torah_submit]
```

## שדות ACF

התוסף רושם אוטומטית קבוצת שדות עבור סוג התוכן "פעילויות עסק תורה":

- סמל מוסד - 6 ספרות.
- שם בית הספר.
- מחוז.
- שלב גיל.
- בנים / בנות / מעורב.
- סוג פעילות.
- פירוט הפעילות.
- צ׳קליסט הכנות.
- 5 שדות תמונה.
- 5 קבצי עזר ו־5 קישורי עזר.
- שותפים.
- ציטוטי משובים.

## אישור תכנים

שליחה מהטופס הקדמי יוצרת פעילות בסטטוס `ממתין לאישור`. עורכים ומנהלים רואים פעמון בסרגל הניהול עם מספר הפעילויות שממתינות לאישור.

## לוגו

התוסף משתמש בלוגו האתר שמוגדר בהתאמה אישית של WordPress. אם אין לוגו מוגדר, התוסף מציג fallback לפי לוגו חמ"ד שסופק.
