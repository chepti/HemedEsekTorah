# Hemed Esek Torah

תוסף WordPress לשיתוף פעילויות "עסק תורה" בחמ"ד.

## דרישות

- WordPress 6.0 ומעלה
- PHP 7.4 ומעלה
- ACF חינמי פעיל באתר

## התקנה

במאגר הזה כל קבצי התוסף נמצאים בתוך התיקייה `hemed-esek-torah/`.

1. להעתיק את **כל** התיקייה `hemed-esek-torah` (עם השם הזה) אל `wp-content/plugins/`, כך שהנתיב לקובץ הראשי יהיה בדיוק:  
   `wp-content/plugins/hemed-esek-torah/hemed-esek-torah.php`
2. לא לשים את התיקייה בתוך תת־תיקייה נוספת תחת `plugins` (אחרת וורדפרס לא ימצא את קובץ ה־PHP בשורש התוסף).
3. להפעיל את התוסף דרך מסך התוספים.
4. להפעיל את ACF אם הוא עדיין לא פעיל.
5. לגשת אל הגדרות > מבנה קישורים וללחוץ שמירה.

אם מופיעה השגיאה **קובץ התוסף לא קיים**: לבדוק בשרת שהנתיב הוא `.../plugins/hemed-esek-torah/hemed-esek-torah.php` ולא `.../plugins/משהו-אחר/hemed-esek-torah/hemed-esek-torah.php` בגלל אריזה כפולה.

## שימוש

בדף הבית מומלץ להכניס את השורטקוד:

```text
[hemed_esek_torah_home]
```

השורטקוד מציג גריד פעילויות, חיפוש וסינון, וכפתור שפותח טופס העלאה בחלונית.

אפשר גם להשתמש בנפרד:

```text
[hemed_esek_torah_grid]
[hemed_esek_torah_submit]
```

## שדות ACF

התוסף רושם אוטומטית קבוצת שדות עבור סוג התוכן "פעילויות עסק תורה":

- סמל מוסד - 6 ספרות.
- שם בית הספר.
- מחוז.
- שלב גיל.
- בנים / בנות / מעורב.
- סוג פעילות.
- פירוט הפעילות.
- צ׳קליסט הכנות.
- 5 שדות תמונה.
- 5 קבצי עזר ו־5 קישורי עזר.
- שותפים.
- ציטוטי משובים.

## אישור תכנים

שליחה מהטופס הקדמי יוצרת פעילות בסטטוס `ממתין לאישור`. עורכים ומנהלים רואים פעמון בסרגל הניהול עם מספר הפעילויות שממתינות לאישור.

## לוגו

התוסף משתמש בלוגו האתר שמוגדר בהתאמה אישית של WordPress. אם אין לוגו מוגדר, התוסף מציג fallback לפי לוגו חמ"ד שסופק.
